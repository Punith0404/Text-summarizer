# Text Summarizer (Streamlit + Groq)

## Project Overview

This project is a web-based text summarization tool built using Streamlit and Groq’s language model API.
It allows users to input large text and generate concise summaries in different formats such as short, medium, and detailed.

---

## Tech Stack

* Python
* Streamlit
* Groq API
* python-dotenv

---

## Problem Statement

The project helps users quickly understand large amounts of text by converting it into shorter, meaningful summaries.
It demonstrates how AI can be used to improve readability and save time when working with lengthy content.

---

## How to Run

### Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Set up environment variables

Create a `.env` file in the root directory and add:

```bash
GROQ_API_KEY=your_api_key_here
```

### Run the application

```bash
streamlit run app.py
```

The application will open in your browser.
